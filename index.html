Context

Universal access to healthcare and well-being for all ages is a UN goal for 2030. Regular health monitoring is essential for prevention and early diagnosis in an ageing society. Current monitoring technologies are fragmented and poorly integrated into daily life, limiting consistent use. There is a need for health monitoring solutions naturally embedded in the home environment.

CasaViva+ & VITALITY

Smart home promoting a healthier and dignified life at home.

  • Assistance in Tasks
  • Health monitoring
  • Prevention and Rehabilitation

Goals

01

Develop a user-centric smart home system to improve convenience, comfort, and functionality.

02

Define system architecture and specifications based on user needs, personas, and existing technologies.

03

Create a flexible, modular infrastructure to integrate multi-device data into a unified semantic framework.

04

Establish an adjustable IoE-based testing environment for realistic evaluation and iterative development.
