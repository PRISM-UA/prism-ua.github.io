Proactive Residential Infrastructure for Smart Monitoring

PRISM◆

A project aimed at promoting essential health services without financial hardship and ensuring well-being for all.

Explore the project →

Context

Regular, proactive health monitoring is crucial for prevention and early diagnosis, empowering individuals—especially in an aging society—to manage their well-being, reduce the burden on healthcare facilities, and live independently for longer. That's what this project is all about.

The aim is to create a sophisticated ICT infrastructure that unifies various devices into a single, intuitive ecosystem, making health monitoring effortless and encouraging long-term adherence.

Key objectives include:

  • Establishing a laboratory for testing
  • Enhancing assistants with robust security and privacy features
  • Progressing monitoring technologies to showcase a model for healthcare at home

Goals

01

Develop a user-centric smart home system to improve convenience, comfort, and functionality.

02

Define system architecture and specifications based on user needs, personas, and existing technologies.

03

Create a flexible, modular infrastructure to integrate multi-device data into a unified semantic framework.

04

Establish an adjustable IoE-based testing environment for realistic evaluation and iterative development.
